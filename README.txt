# .NET Starter

Bluemix provides a .NET starter application as a template so that you can add your code and push the changes back to the Bluemix environment.


## Files

The application has files as below:

*   NuGet.Config

	NuGet package manager configuration settings.

*   dotnetstarter.sln

	Visual Studio solution file.

*   src/samplemvc/project.json

	Lists the applications dependencies.

*   src/samplemvc/Startup.cs

	Configures the ASP.NET request pipeline.

*   wwwroot/

	Static files like images and CSS.

*   Views/

	HTML files for displaying the application.
